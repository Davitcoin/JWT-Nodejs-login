## Description

Sign up and login with jwt node application using nestjs and prisma

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

#Prisma
$ npx prisma studio

```
